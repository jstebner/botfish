## Getting Started

### Dependencies

* REVIEW: Requires Python 3.10 on Windows 10/11, Ubunutu 22 
* Packages listed in botfish/requirements.txt

### Installing

```
REVIEW:
git clone https://github.com/jstebner/botfish.git
cd botfish
py 3.10 -m pip install -r requirements
```

### Usage

```
py ./botfish.py [args]
```
